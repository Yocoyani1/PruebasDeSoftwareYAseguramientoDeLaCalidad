# P2 – Resultados / Evidencia

En esta carpeta se guarda la evidencia de corridas exitosas del programa **convertNumbers** (P2).

- **ConvertionResults.txt**: salida generada (números en binario y hexadecimal, tiempo de ejecución).
- **Capturas de pantalla**: se aceptan capturas de los resultados exitosos en lugar de un PDF.

## Cómo generar evidencia

Desde la raíz del repositorio:

```bash
cd P2/source
python convertNumbers.py ../../data/numbers.txt ../results
```

El archivo `ConvertionResults.txt` se creará en `P2/results/`.
